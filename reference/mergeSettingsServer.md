# Merge Settings UI

UI of the merge settings module

Server function of the merge settings module

## Usage

``` r
mergeSettingsUI(id)

mergeSettingsServer(id, tableXData, tableYData, tableXId, tableYId)
```

## Arguments

- id:

  id of module

- tableXData:

  (data.frame) data to be merged

- tableYData:

  (data.frame) data to be merged

- tableXId:

  (character) internal table name, see
  [extractTableIds](https://pandora-isomemo.github.io/data-tools/reference/extractTableIds.md)

- tableYId:

  (character) internal table name, see
  [extractTableIds](https://pandora-isomemo.github.io/data-tools/reference/extractTableIds.md)
