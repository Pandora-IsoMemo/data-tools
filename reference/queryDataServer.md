# Query Data UI

UI of the query data module

Server function of the qery data module

## Usage

``` r
queryDataUI(id)

queryDataServer(id, dataProcessList, isActiveTab)
```

## Arguments

- id:

  id of module

- dataProcessList:

  (list) list of data to be merged

- isActiveTab:

  (reactive) TRUE if tab of query module is the active
