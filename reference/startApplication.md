# Start Application

Start Application

## Usage

``` r
startApplication(
  port = getOption("shiny.port"),
  host = getOption("shiny.host", "127.0.0.1"),
  launch.browser = getOption("shiny.launch.browser", interactive())
)
```

## Arguments

- port:

  port

- host:

  host

- launch.browser:

  If true, the system's default web browser will be launched
