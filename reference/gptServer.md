# GPT UI

UI of the gpt module

Server function of the gpt module

## Usage

``` r
gptUI(id)

gptServer(id, autoCompleteList, isActiveTab)
```

## Arguments

- id:

  id of module

- autoCompleteList:

  (list) word to be used for auto completion

- isActiveTab:

  (reactive) TRUE if tab of query module is the active
